/******************************
** Installation Instructions **
******************************/
0. Install fonts from Folder.
1) Unzip the file NSA_Conky.zip
2) Copy or Move the folder [NSA_Conky] in the subdirectory /home/your_user_name/conky-manager/themes/
   (short way: ~/conky-manager/themes/)
3) Start the Conky manager and enabled NSA_Conky theme
